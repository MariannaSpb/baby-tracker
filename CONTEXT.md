# Baby Tracker — Project Context

## What is this
A personal web app for tracking newborn sleep, feeds, and diapers.
Used by one parent on mobile browser, one-handed, often at night.

## User context
- Sleep-deprived parent
- One hand free (holding baby)
- Low light conditions at night
- Needs large tap targets and high contrast

## Design
- Mobile-first, max-width 430px
- Light background, soft colors
- Large touch targets (min 44x44px)
- Clean, minimal UI

## Stack
- Angular 21, standalone components, SCSS
- No UI library — custom components only

## Components to build
- Dashboard (main screen with quick action buttons)
- Feed logger
- Sleep logger  
- Diaper logger
